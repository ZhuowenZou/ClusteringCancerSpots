This code does clustering on the dataset presented under the folder.

Dependency:
Please install the latest version of python, jupyter notebook, sklearn, matplotlib, and numpy.

To run:
In terminal, run "jupyter notebook clusterer.ipynb"


File explanation:
clusterer.ipynb: the main file for doing clustering, manipulate results, and plot graphs.
Dataset.xlsx: the data that's analyzed
util.py: some utility function, including loading the data.
